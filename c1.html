unelte de modelat si olarit

  1. cutit si modelator convex din lemn
    2. ustensila multifunctionala, ajuta la stabilirea formei necizelate
  2. cutit mare blunt
    3. cu el taien blocul de lut
  3. ata
    4. cu ea scoatem obiecul din lut de pe roata sau suport
  4. lama mare
    5. indreapta peretii vasului
  5. instrumente laso
    6. extrag materialul extra de pe pereti
  6. laso mare tuguiat
    7. aceiasi functie, acopera diferit suprafata
  7. cutit de precizie
    8. taieri mici, cu importanta la detaliu
  8. burete
    9. absoarbe lichid extra de pe lucrare
  9. set de utilitate
    10. ajuta la formarea detaliilor/ partilor functionale specifice
  10. periuta cilindrica
    11. curata cavitatile
  11. periuta metalica
    12. creaza textura

tipuri de lut

  1. lut cu uscare la aer
    • greu de folosit, se lipeste de mana, necesita apa
    • relativ accesibil
    • potrivit pentru decor
    • necesita expeienta/cunostinte de lucru
    • crapa
    • este greu adica instabil
    • necesita timp de uscat
    • necesita conditii speciale de pastrare
  2. lut polimeric
    • necesita cuptor simplu
    • este foarte curat
    • este usor de folosit
    • vine pre-colorat
    • este FOARTE scump
    • are risc de a crapa
    • este suportabil de usor
    • nu necesita pastrare speciala**
    • se vinde si se foloseste in cantitati mici
  3. plastilina pentru modelat
    • este foarte usuoara
    • murdareste tare
    • nu este permanenta
    • este sensibila la lumina si caldura
    • nu necesita experienta
    • este ieftina
    • se lipeste bine de sine
    • poate fi reutilizata
